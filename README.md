# Anmmaa Group Website

A simple static website showcasing Anmmaa Group and its subsidiaries.

## Overview

This website provides an overview of Anmmaa Group, the parent company, and details about each subsidiary including their services and contact information where available.

## Files

- `index.html`: The main HTML page.
- `styles.css`: Basic CSS styling for the website.

## How to View

Open `index.html` in any web browser to view the website.

## Technologies Used

- HTML5
- CSS3

For more information, refer to the subsidiary details on the website.
